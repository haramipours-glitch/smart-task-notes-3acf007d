# راهنمای دانلود APK از Codemagic

## مرحله ۱: ورود به Codemagic

۱. به سایت برو:
   ```
   https://codemagic.io
   ```

۲. با GitHub وارد شو.

## مرحله ۲: پیدا کردن Build

۱. پروژه `smart-task-notes-3acf007d` را باز کن.

۲. به تب **Build history** یا **Builds** برو.

۳. آخرین build را پیدا کن که سبز (✅) است.
   - نام آن باید `android-debug-apk` باشد.

## مرحله ۳: دانلود APK

### روش اول: از داخل build

۱. روی build سبز کلیک کن.

۲. صفحه جزئیات build باز می‌شود.

۳. به پایین صفحه برو.

۴. بخش **Artifacts** را ببین.

۵. فایل `app-debug.apk` را پیدا کن.

۶. روی آن کلیک کن تا دانلود شود.

### روش دوم: از ایمیل

اگر در Codemagic ایمیل را تنظیم کرده باشی، بعد از build موفق، یک ایمیل با لینک دانلود APK می‌فرستد.

## مرحله ۴: انتقال به گوشی

۱. APK را با کابل USB به گوشی منتقل کن.

۲. یا با Telegram/WhatsApp/Bluetooth به گوشی بفرست.

۳. روی گوشی، APK را باز کن.

۴. اگر پیام "نصب از منابع ناشناخته" داد، در تنظیمات اجازه بده.

۵. **Install** را بزن.

## مرحله ۵: اضافه کردن ویجت

۱. برنامه را یک بار باز کن.

۲. روی صفحه اصلی گوشی نگه دار.

۳. **Widgets** را انتخاب کن.

۴. **ARSHNAZ** را پیدا کن و به صفحه اضافه کن.

## ❌ اگر مشکل داری

- اگر نمی‌توانی APK را دانلود کنی، متن خطا یا عکس بفرست.
- اگر build قرمز است، نام مرحله قرمز و متن خطا را بفرست.
