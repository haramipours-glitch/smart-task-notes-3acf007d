# ARSHNAZ Android Widget

این ویجت برای برنامه ARSHNAZ طراحی شده است تا کاربران بتوانند بدون باز کردن برنامه، تسک‌های خود را ببینند و تسک جدید اضافه کنند.

## ویژگی‌ها

- نمایش تعداد تسک‌های امروز (placeholder - در نسخه بعدی از Supabase واقعی)
- نمایش تاریخ به شمسی
- دکمه "+ تسک جدید" برای افزودن سریع تسک
- کلیک روی ویجت برای باز کردن برنامه و رفتن به صفحه امروز
- تم تاریک هماهنگ با برنامه
- اندازه قابل تنظیم (resizeable)

## نصب

۱. برنامه را روی اندروید نصب کنید.
۲. روی صفحه اصلی، طولانی بزنید تا ویجت‌ها نمایش داده شوند.
۳. ویجت ARSHNAZ را انتخاب کنید و به صفحه اصلی اضافه کنید.

## نحوه کار

- **تسک جدید:** دکمه "+ تسک جدید" را بزنید تا صفحه افزودن تسک باز شود.
- **باز کردن برنامه:** روی هر جای ویجت بزنید تا برنامه باز شود و به صفحه امروز برود.
- **به‌روزرسانی:** ویجت هر ۳۰ دقیقه خودکار به‌روز می‌شود.

## توسعه آینده

- اتصال به Supabase برای نمایش تعداد واقعی تسک‌ها
- نمایش تسک‌های مهم با رنگ‌بندی
- افزودن دکمه‌های سریع برای Pomodoro و Check-in
- پشتیبانی از چند اندازه ویجت
