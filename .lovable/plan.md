## بخش ۱ — بهینه‌سازی بصری صفحه‌ی Task

وضعیت فعلی: سه بلاک «دسته‌بندی / زمان‌بندی / خرد کردن کار» با border + بک‌گراند `bg-muted/20` + پدینگ، هرکدوم Collapsible. در موبایل هنوز سنگین و پر از خط دیده می‌شه.

تغییرات پیشنهادی (فقط UI، بدون تغییر منطق):

1. **حذف کادرها، استفاده از divider نازک**
   - حذف `border + rounded-2xl + bg-muted/20` از سه `<section>`.
   - جداکننده‌ی نازک افقی (`border-t border-border/40`) بین بلاک‌ها به‌جای کارت‌های توی هم.
   - نتیجه: ظاهر مثل صفحات تنظیمات iOS، خیلی سبک‌تر.

2. **هدر بلاک‌ها مینیمال‌تر**
   - حذف نوار رنگی کنار عنوان (`<span className="w-1 h-3.5 bg-primary/60" />`).
   - عنوان: کوچک‌تر، رنگ `text-muted-foreground/70`، بدون uppercase.
   - فقط chevron در سمت چپ.

3. **اولویت به‌صورت چیپ افقی بدون کادر تو در تو**
   - حذف `rounded-lg border bg-background` دور بخش اولویت.
   - چهار چیپ کوچک افقی نمایش مستقیم اولویت‌ها (active = پر، بقیه = ghost). دیگه نیازی به اکاردئون داخل اکاردئون نیست.
   - تیک «اجتنابی» به‌صورت Switch کوچک سمت راست.

4. **Folder + Tags سبک‌تر**
   - تبدیل از دو دکمه‌ی `outline h-9` به دو ردیف ساده با آیکن + متن + شِوْرِن، شبیه ردیف‌های لیست.

5. **Time Block سبک‌تر**
   - حذف رنگ پررنگ `border-primary/30 bg-primary/5`. فقط یک ردیف ساده با badge کوچک «فعال» وقتی دیتا داره.

6. **Spacing کلی**
   - `space-y-5` بین بلاک‌ها → `space-y-0` با divider.
   - پدینگ افقی بلاک‌ها در موبایل `px-0` (چون divider full-width قشنگ‌تره).

7. **تیتر تسک**
   - حذف کارت `border bg-card/40 shadow-sm` دور تیتر و توضیحات. فقط متن خالص با divider زیرش.

نتیجه‌ی نهایی: کمتر از نصف خطوط visual فعلی، حس Apple Notes / Things 3.

## بخش ۲ — یادآور (سؤال کاربر)

**پاسخ صادقانه:** الان یادآور تسک‌ها **کار نمی‌کنه**. در `src/lib/reminders.ts` فقط «چک‌این روزانه» چک می‌شه. فیلد `reminder_at` هر تسک در دیتابیس ذخیره می‌شه ولی هیچ‌جا خونده نمی‌شه و هیچ نوتیفیکیشن/رنگی روی تسک ظاهر نمی‌شه.

پیشنهاد فعال‌سازی:

1. **افزودن `checkTaskReminders(userId)` به reminders.ts**
   - هر ۶۰ ثانیه (هم‌زمان با تیک موجود) تسک‌هایی که `reminder_at <= now` و `completed = false` و هنوز fire نشده‌اند رو پیدا کنه.
   - برای هرکدوم `new Notification(...)` با عنوان تسک.
   - برای جلوگیری از تکرار: ذخیره‌ی id های fire شده در localStorage (`reminder_fired_tasks_v1`).
   - صدای کوتاه اختیاری با Web Audio.

2. **نشان بصری روی تسک (رنگی)**
   - در لیست تسک‌ها (`TasksView` / کارت تسک)، اگر `reminder_at` گذشته و تسک کامل نشده → یک نقطه‌ی قرمز پالس‌دار کنار آیکن زنگوله.
   - اگر `reminder_at` در آینده‌ی نزدیک (< ۱ ساعت) → رنگ amber.

3. **اطمینان از permission**
   - وقتی کاربر اولین بار `reminder_at` تنظیم می‌کنه، `ensureNotificationPermission()` صدا زده بشه و اگر رد شد توست راهنما.
